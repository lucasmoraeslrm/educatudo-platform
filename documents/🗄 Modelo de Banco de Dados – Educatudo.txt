📌 Autenticação por Perfil
* Admin Global → email + senha

* Admin Escola (Coordenação) → email + senha

* Professor → codigo_prof + senha

* Aluno → ra + senha

* Pais → email + senha

________________


📊 Resumo:
   * usuarios → tabela central de autenticação.

   * alunos, professores e pais → tabelas específicas com informações adicionais.

   * Login diferenciado: RA (aluno), Código (professor), E-mail (admin/pais).

   * Isolamento: todos os registros vinculados a escola_id.
